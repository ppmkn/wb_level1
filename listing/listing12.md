Что выведет данная программа и почему?

```go
  func main() {
  n := 0
  if true {
  n := 1
  n++
  }
  fmt.Println(n)
  }
```

**Ответ:**

`0`

Дело в области видимости. После завершения блока `if`, переменная `n`, объявленная внутри него, выходит из области видимости, 
и нам остается доступной переменная `n` из блока `main`.
